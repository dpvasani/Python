## ğŸ“Œ What is `enumerate()`?

`enumerate()` is a **built-in Python function** that lets you **loop through an iterable (like a list or tuple)** and **track the index** of each item **at the same time** â€” all without writing extra code. ğŸ§™â€â™‚ï¸

## ğŸ§  Why Use `enumerate()`?

Without `enumerate()`:

## ğŸ§ª Syntax

* `iterable` â€“ any iterable object (list, tuple, string, etc.)

## ğŸ” `json.load()`: Read JSON from a File

> ğŸ“– Think of `json.load()` as **loading structured JSON data from a `.json` file into a Python object** (like dict or list).

### ğŸ§  Syntax:


### âœ… Example:

Suppose you have a file `data.json`:

```json
  "age": 23,
  "skills": ["Python", "React", "Node"]
```

### ğŸ§ª Python Code to Read:

```python

```

### ğŸ” What You Get:

A Python dictionary!

---

## ğŸ“ `json.dump()`: Write Python Data to a JSON File

> ğŸ’¾ Use `json.dump()` to **serialize** Python objects (like dict, list) into a file as JSON format.

### ğŸ§  Syntax:

### âœ… Example:

### ğŸ“‚ What you get in `output.json`:

### ğŸ¨ `indent=4` makes it **pretty printed**!

---

## ğŸ¤” When to Use What?

| Task                            | Use                     |
| ------------------------------- | ----------------------- |
| âœ… Read from `.json` file        | `json.load(file)`       |
| âœ… Write to `.json` file         | `json.dump(data, file)` |
| âœ… Convert JSON string to Python | `json.loads(string)`    |
| âœ… Convert Python to JSON string | `json.dumps(data)`      |

---

## âš ï¸ Common Mistakes

âŒ **Using `json.load()` on a JSON string**
âœ… Use `json.loads()` instead.

âŒ **Writing JSON to file using `dumps()`**
âœ… Use `json.dump()` for writing directly to a file.
