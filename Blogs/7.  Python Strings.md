## üìå 1. What is a String?

A **string** in Python is a **sequence of characters** enclosed in **single quotes (' ')**, **double quotes (" ")**, or **triple quotes (''' or """)** for multi-line text.

```python
str1 = 'Hello'
str2 = "World"
str3 = '''Triple
quoted string'''
```

---

## üîç 2. String Basics

| Concept     | Syntax/Example   | Notes                     |
| ----------- | ---------------- | ------------------------- |
| Declare     | `s = "hello"`    | Strings are immutable     |
| Access      | `s[0]`           | Indexing starts at 0      |
| Slice       | `s[1:4]`         | Slicing is `[start:end]`  |
| Length      | `len(s)`         | Counts characters         |
| Concatenate | `"hi" + "bye"`   | Joins two strings         |
| Repeat      | `"ha" * 3`       | Returns `"hahaha"`        |
| Membership  | `'h' in 'hello'` | Returns `True` or `False` |

---

## ‚úÇÔ∏è 3. Slicing & Indexing üéØ

```python
s = "Python"
s[0]     # 'P'
s[-1]    # 'n'
s[0:2]   # 'Py'
s[:3]    # 'Pyt'
s[2:]    # 'thon'
s[::-1]  # 'nohtyP' (reverse string)
```

# üîç Part 1: Regular Expressions (Regex) in Python

## üéØ What is Regex?

> Regular Expressions (Regex) are **patterns** used to **match strings** or **substrings**.

Python has a built-in module:

```python
import re
```

---

## üß† Common Regex Patterns

| Pattern | Meaning                      | Example Match                      |       |       |
| ------- | ---------------------------- | ---------------------------------- | ----- | ----- |
| `.`     | Any character except newline | `a`, `b`, `1`, `!`                 |       |       |
| `^`     | Start of string              | `^Hello` ‚Üí matches `"Hello World"` |       |       |
| `$`     | End of string                | `end$`                             |       |       |
| `*`     | 0 or more occurrences        | `a*` ‚Üí `""`, `"a"`, `"aaa"`        |       |       |
| `+`     | 1 or more                    | `a+` ‚Üí `"a"`, `"aaa"`              |       |       |
| `?`     | 0 or 1                       | `a?` ‚Üí `""`, `"a"`                 |       |       |
| `[]`    | Any one of the characters    | `[aeiou]`                          |       |       |
| `[^]`   | Not in set                   | `[^aeiou]`                         |       |       |
| `{n}`   | Exactly n times              | `a{3}` ‚Üí `"aaa"`                   |       |       |
| `{n,}`  | n or more                    | `a{2,}` ‚Üí `"aa"`, `"aaa"`          |       |       |
| `{n,m}` | Between n and m times        | `a{1,3}` ‚Üí `"a"`, `"aa"`           |       |       |
| \`      | \`                           | OR                                 | \`cat | dog\` |
| `\d`    | Digit (0‚Äì9)                  | `"123"`                            |       |       |
| `\D`    | Non-digit                    | `"abc"`                            |       |       |
| `\w`    | Word character (a-zA-Z0-9\_) | `"hello_123"`                      |       |       |
| `\W`    | Non-word character           | `"@! "`                            |       |       |
| `\s`    | Whitespace                   | `" "`                              |       |       |
| `\S`    | Non-whitespace               | `"abc"`                            |       |       |

---

## üß∞ Common Functions in `re` module

```python
import re

# ‚úÖ Search pattern in string
re.search("pattern", "text")

# ‚úÖ Match only from start
re.match("Hello", "Hello World")

# ‚úÖ Find all matches
re.findall("\d+", "a1 b22 c333")  # ['1', '22', '333']

# ‚úÖ Substitute
re.sub("\s+", "-", "hello   world")  # 'hello-world'

# ‚úÖ Compile for reuse
pattern = re.compile("[a-z]+")
pattern.findall("abc XYZ 123")  # ['abc']
```

## üî• Harder Level

9. **Compress string (run-length encoding)**
   `"aaabbccc"` ‚Üí `"a3b2c3"`

```python
def compress(s):
    res = ""
    count = 1
    for i in range(1, len(s)):
        if s[i] == s[i-1]:
            count += 1
        else:
            res += s[i-1] + str(count)
            count = 1
    res += s[-1] + str(count)
    return res
```

10. **Check if rotation**
    `"abc"` and `"cab"` ‚Üí ‚úÖ

```python
def is_rotation(a, b):
    return len(a) == len(b) and b in a + a
```

11. **Group anagrams**

```python
from collections import defaultdict
def group_anagrams(words):
    d = defaultdict(list)
    for w in words:
        d[''.join(sorted(w))].append(w)
    return list(d.values())
```

## üì¶ Bonus Challenge

12. **Valid Parentheses**

```python
def is_valid(s):
    stack = []
    mapping = {')':'(', ']':'[', '}':'{'}
    for ch in s:
        if ch in mapping.values():
            stack.append(ch)
        elif ch in mapping:
            if not stack or mapping[ch] != stack.pop():
                return False
    return not stack
```
